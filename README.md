# 🧮 LaTeX to Unicode Telegram Userbot

Умный и быстрый Userbot для Telegram, который **автоматически** конвертирует LaTeX-подобные команды в красивые Unicode-символы прямо в момент отправки или редактирования сообщения. Никаких дополнительных команд или префиксов писать не нужно!

## ✨ Возможности

- **🪄 Полная автоматизация:** Просто напишите `\alpha^2`, и бот мгновенно заменит это на `α²`.
- **📐 Математика и логика:** `+`, `-`, `=`, `\times`, `\div`, `\infty`, `\neq`, `\leq`, `\sum`, `\int`, `\forall`, `\implies` и многие другие.
- **🔤 Греческий алфавит:** Строчные (`\alpha`, `\beta`, `\gamma`) и прописные (`\Gamma`, `\Delta`, `\Omega`) буквы.
- **🌌 Астрономия:** Символы Солнца (`\sun` ☉), Луны (`\moon` ☽), Земли (`\earth` ♁) и всех планет (`\mercury` ☿, `\venus` ♀, `\mars` ♂, `\jupiter` ♃, `\saturn` ♄, `\uranus` ♅, `\neptune` ♆, `\pluto` ♇).
- **⚛️ Физика и химия:** `\hbar` (ℏ), `\angstrom` (Å), `\ohm` (Ω), `\celsius` (℃).
- **⬆️⬇️ Индексы:** Поддержка верхних (`^2`, `^{abc}`) и нижних (`_2`, `_{xyz}`) индексов для цифр, латиницы и отдельных символов латиницы/кириллицы/греческого.
- **✏️ Редактирование:** Конвертация срабатывает не только при отправке нового сообщения, но и когда вы его редактируете.

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/latex-telegram-bot.git
cd latex-telegram-bot
```

### 2. Создание виртуального окружения
```bash
python3 -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```
> **⚠️ Примечание для Linux:** Если при установке `tgcrypto` возникает ошибка `Python.h: No such file`, установите пакеты разработчика:  
> `sudo apt update && sudo apt install python3-dev build-essential`  
> *(Либо просто удалите строку `tgcrypto==1.2.5` из `requirements.txt` — бот будет работать чуть медленнее, но без проблем).*

### 4. Получение ключей API
1. Перейдите на [my.telegram.org](https://my.telegram.org) и войдите в аккаунт.
2. Выберите **API development tools**.
3. Создайте новое приложение (название и короткое имя могут быть любыми).
4. Скопируйте **App api_id** (число) и **App api_hash** (строка).

### 5. Настройка конфигурации
Создайте файл `.env` в корневой папке проекта (или переименуйте `.env.example`):
```env
# Обязательные параметры
API_ID=12345678
API_HASH=ваш_длинный_api_hash_здесь

# Необязательные параметры
# SESSION_STRING=ваша_строка_сессии_если_хотите_входить_без_смс
PREFIX=.

# НАСТРОЙКИ ПРОКСИ (Раскомментируйте и заполните, если Telegram блокируется вашим провайдером)
# PROXY_HOSTNAME=127.0.0.1
# PROXY_PORT=1080
# PROXY_SCHEME=socks5
# PROXY_USERNAME=
# PROXY_PASSWORD=
```

### 6. Запуск
```bash
python main.py
```
При первом запуске (если не указан `SESSION_STRING`) бот попросит ввести ваш номер телефона и код подтверждения из Telegram. После этого в папке появится файл `latex_userbot.session`.

## 💡 Руководство по использованию

Просто начните печатать сообщение в Telegram как обычно. Если бот обнаружит в тексте символы `\`, `^` или `_`, он обработает текст и **отредактирует** ваше сообщение, заменив команды на символы.

### Примеры

| Вы пишете | Бот отправляет |
| :--- | :--- |
| `Скорость света c \approx 3 \times 10^8 м/с` | Скорость света c ≈ 3 × 10⁸ м/с |
| `Формула воды: H_2O, угол \theta = 90\degree` | Формула воды: H₂O, угол θ = 90° |
| `\sun > \earth > \moon` | ☉ > ♁ > ☽ |
| `Пусть x \in \mathbb{R}, \alpha \neq 0` | Пусть x ∈ ℝ, α ≠ 0 |
| `\frac{1}{2} + \frac{3}{4} = \frac{5}{4}` | ½ + ¾ = ⅝ |

> **💡 Лайфхак:** Если вам нужно отправить обычный обратный слеш (например, путь к файлу `C:\Windows`), используйте двойной слеш `\\`. Бот превратит его в одинарный `\` и не будет искать команду.

## 🛠 Расширение функционала

Хотите добавить свои символы? Это очень просто! Откройте файл `utils/latex_converter.py` и добавьте новые пары ключ-значение в словари:
- `SUPERSCRIPT_MAP` (для верхних индексов, например, `'q': 'ᵠ'`)
- `SUBSCRIPT_MAP` (для нижних индексов)
- `LATEX_COMMANDS` (для именованных команд, например, `r'\my_symbol': '★'`)

Скрипт автоматически подхватит новые значения без изменения основной логики.

## 🔧 Решение частых проблем (Troubleshooting)

### 1. Ошибка `Connection timed out` при запуске
Это означает, что ваш интернет-провайдер блокирует прямое соединение с серверами Telegram.
**Решение:** 
1. Включите VPN на вашем устройстве.
2. Узнайте локальный порт SOCKS5 прокси вашего VPN-клиента (обычно `1080`, `10808` или `7890`).
3. Раскомментируйте и заполните строки `PROXY_*` в файле `.env`.
4. Перезапустите бота.

### 2. Ошибка `MESSAGE_ID_INVALID` (если вы пытались использовать отложенные сообщения)
Текущая стабильная версия бота оптимизирована для мгновенной конвертации новых и редактируемых сообщений. Функция работы с отложенными (запланированными) сообщениями была удалена из-за ограничений API Telegram, требующих сложной синхронизации времени, которая часто приводила к сбоям.

## 🛡️ Безопасность (ОЧЕНЬ ВАЖНО)

1. **Никогда не загружайте файл `.env` в публичные репозитории (GitHub, GitLab).** В нем содержатся ключи доступа к вашему аккаунту. Файл `.gitignore` уже настроен на игнорирование `.env`.
2. **Файл `*.session` (например, `latex_userbot.session`) равносилен паролю от вашего аккаунта.** Любой, кто получит этот файл, сможет войти в ваш Telegram. Храните его в секрете и не передавайте третьим лицам.
3. Если вы подозреваете, что данные скомпрометированы, немедленно зайдите в **Настройки Telegram → Конфиденциальность → Активные сеансы** и завершите подозрительные сеансы.
